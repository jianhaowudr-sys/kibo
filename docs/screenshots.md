# 上架截圖清單（依 App Store + Play Store 必備）

兩平台都各需 5-8 張截圖。以下情境每張都要拍（**iOS 一定要 6.7" iPhone 15/16 Pro Max 解析度 1290×2796**；Android 隨便手機 9:16 即可）。

## 必拍 8 張（按權重排序）

### 1. 首頁 — 蛋 + 連續日 + 動圈
- 蛋的 EXP 進度條清楚可見
- 至少 2-3 天連續日 streak
- 「今日三件事」卡片可見

### 2. 飲食 AI 拍照辨識結果
- 顯示一個拍好的便當照
- AI 辨識出的 items 清單（雞腿/白飯/蔬菜各別標出）
- 熱量 + 蛋白質 + 碳水 + 脂肪總計

### 3. 進行中訓練畫面
- 訓練時間倒數 + EXP 累積
- 動作清單 chip（3-4 個動作）
- 中間正在輸入的某動作有 2-3 組已完成
- 組間計時 active 時的大字倒數（最好截到這狀態）

### 4. 健康四件事卡片組
- 喝水 75% / 排便 / 睡眠 / 月經 四張並排
- 在首頁狀態

### 5. 寵物頁面
- 你孵出的像素寵物 + 名字
- 等級 / EXP / 餵食按鈕

### 6. 體組成 InBody 趨勢
- 體重 / 體脂 / 骨骼肌曲線圖
- 至少 5-7 個資料點

### 7. 食物庫
- 5-10 個常吃食物的 row
- 顯示 emoji + 名稱 + 份量 + 熱量

### 8. 設定頁（呈現完整功能感）
- 滑到中間，能同時看到「個人」「帳號」「App」群組標題

## 拍攝技巧
- 用「真機」不要用模擬器（陰影、字體渲染不同）
- 開「夜晚模式」截 4 張、「白天模式」截 4 張，輪替放（看起來支援深色）
- iOS 要先把 Status Bar 時間調成 9:41（Apple 慣例）
- Android 不用調 Status Bar
